## 📁 Repository Structure

Each problem is organized in its own dedicated directory with the following structure:

```
problem-number-problem-title/
├── README.md              # Problem description, constraints & approach
├── solution.cpp           # C++ optimized solution
├── solution.js            # JavaScript implementation
└── Notes.md               # Additional insights & tips (if applicable)
```

**Example:** `1-two-sum/solution.cpp`

> **Naming Convention:** Problems follow the format `{problem-id}-{problem-title}/`

## 🚀 Installation & Setup

### Prerequisites
- Git installed on your machine
- C++ compiler (g++/clang) for running C++ solutions
- Node.js (optional) for running JavaScript solutions

### Clone Repository

```bash
git clone https://github.com/harshitj183/Leetcode.git
cd Leetcode
```

### Compile C++ Solution (Example)

```bash
cd 1-two-sum
g++ -o solution solution.cpp
./solution
```

### Run JavaScript Solution

```bash
node solution.js
```
